Главный проект

Supplier Finder AI

В рамках проекта Supplier Finder AI реализовал полный AI-пайплайн для поиска и оценки поставщиков. Система собирает информацию из открытых источников, обогащает данные, анализирует поставщиков с помощью Gemini и формирует ранжированный результат для пользователя.

Смотреть репозиторий →

Проблема

Поиск поставщиков занимает много времени. Информация разбросана по сайтам, а оценка надежности требует ручного анализа.

Решение

Сервис автоматически ищет поставщиков, собирает данные и использует Gemini AI для оценки и ранжирования результатов.
